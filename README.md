# OneButtonGlow

Unified golden proc glow for all action buttons in World of Warcraft.

## Features

- **Unified Proc Glow**: Replaces various highlight styles with a consistent golden glow.\n- **Action Button Support**: Works across all standard action bars.\n- **Visual Consistency**: Provides a modern and clean look for ability procs.

## Installation

1. Download the latest version.
2. Extract the files to your `Interface/AddOns` folder.
3. Restart World of Warcraft.
